What's new in MPDR v0.3

These are the new features for MPDR version 0.3 and their corresponding subversions.

Version 0.3

New RNG algorithm

Version 0.3 uses a similar random number generator algorithm to syncathetic's original algorithm, so seeds will now produce different results.

Compatibility with other randomizers (Stacking)

From this version, you can use an already modified/randomized image of Metroid Prime, allowing for a combination of another randomizer and MPDR. This is called Stacking.

Currently, there is support only for BashPrime's randomizer version 2.0.

Map patch

Now MPDR is able to edit the world map so that doors appear in the correct type. This can be disabled in the Patches configuration window.

NOTE: Doors with missile shields will be represented in the same color as their door, and not grey like in the original game!

One-way door randomization

The one-way door that connects the Vault to the Main Plaza in Chozo Ruins can now be made two-way and in turn, randomized.

To do so, enable the experimental feature "One-way door randomization" in the Patches configuration window, and in the Door Excluder, enable the randomization on that door.

Bug fixes

  • Doors will not have a slim chance of becoming blue if the door weight for blue doors is set to 0 anymore.
