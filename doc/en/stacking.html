Stacking: Support for modified games

Since version 0.3, MPDR now supports Metroid Prime modified images. The process of randomizing an already modified game is called stacking.

To stack a randomized game with MPDR, you simply select it as the input ISO and MPDR should be able to recognize the modifications made.

Depending on the tools for making these modified images, MPDR will apply, import from the modified game, or ignore certain patches.

Patches compatibility

This table indicates which patches are available for each one of the officially supported randomizers in the latest version of MPDR:

  • ✓ Yes: You can use MPDR to apply this patch on a modified game.
  • ◇ Imported: MPDR will use the patch from the modified game.
  • - Ignored: MPDR will not apply this patch on a modified game.
  • ! Risky: Either MPDR may not be able to function correctly with this patch applied, or the game might have a high chance of crashing, both depending on the settings of the modified game,
  • ⨯ No: MPDR will definitely not be able to function correctly with this patch enabled, or crashes the game constantly.
