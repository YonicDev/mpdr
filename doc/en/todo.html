Coming soon in MPDR

This is a list of the planned features for the upcoming versions. (It also serves as a TODO for the developers!)

They are not ordered in any particular order.

Version support

  • Support for Metroid Prime v1.01 (NTSC 0-01).
  • Support for PAL version.

Randomizer

  • Super Metroid Doors: When a non-blue door is opened, it will turn to blue permanently.

MPDR

  • Full multi-language support: Allows switching between languages without changing the system languages.
  • Graphical Weight Editor: This uses gradient-like editors to modify the door weights.
  • Door Excluder 2.0: Instead of just excluding doors from being randomized, also allow to force them being a different color.
