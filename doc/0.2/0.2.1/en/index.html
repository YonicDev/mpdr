Metroid Prime Door Randomizer v0.2.1

What's new

  • MPDR now supports Mac OS from High Sierra (10.13) onward.
  • Power Bomb lockpicking: When enabled, you may use a Power Bomb to open a door of any color.
  • The ventilation system doors inside the Crashed Frigate can now be randomized.

For more information on the program, check the relevant documentation in 0.2.

This documentation is for MPDR version 0.2.1.
