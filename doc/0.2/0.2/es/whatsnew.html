Novedades de MPDR v0.2

Estas son las principales características añadidas a MPDR version 0.2.

Para ver las características añadidas en versiones menores, véase la documentación para la revisión correspondiente.

Excluir puertas

Ya es posible excluir puertas usando la herramienta de exclusión de puertas, el cual permite permitir o impedir que MPDR randomice las puertas marcadas como excluidas. De esta forma, las puertas excluidas serán del mismo tipo que en el juego original.

Esta herramienta es experimental y no se ha testeado del todo, por lo que puede haber algunos fallos.

Compatibilidad de perifles de MPDR

Los perfiles MPDR v0.2 son de versión 3, pero MPDR es compatible con perfiles de versión 2, y a partir de ahora, de versión 1.

Para convertir un perfil existente a la actual, cárgalo en MPDR y guárdalo con el mismo nombre.

Documentación mejorada

Desde esta versión, MPDR utiliza Chromium en todas sus plataformas para mostrar la documentación, permitiendo una documentación mucho más detallada y fácil de usar.

También permite soporte multilingüe, y la posibilidad de ver la documentación de versiones anteriores si está disponible en tu idioma.
