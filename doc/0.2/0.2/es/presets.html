Perfiles de configuración

Puedes guardar ajustes predefinidos (perfiles) para uso en casos futuros.

Para eso, utiliza el menú Archivo en la parte superior de la ventana principal de MPDR.

MPDR está diseñado para ser compatible con perfiles de versiones anteriores del programa. Para convertir un perfil a la versión actual, simplemente tienes que abrirlo en MPDR y guardarlo de nuevo.
