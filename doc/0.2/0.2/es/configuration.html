Configuring MPDR

You can configure how MPDR randomizes the doors by modifying the following properties.

Basic properties

These properties can be modified from the main window.

Seed

The seed is a number from 0 to 4294967295 (that's 232-1) that the randomizer uses to determine which type any given door will be.

You can type it yourself, have it generated randomly with the Generate button, or convert the current time you're at into a seed number with the Use time button.

By default, it uses the time at which you've opened the program as seed.

Door weights

Door weights modify how frequent a door of a certain type is in one of the five major areas of Tallon IV (Tallon Overworld, Chozo Ruins, Magmoor Caverns, Phendrana Drifts and Phazon Mines).

Each are controlled with a slider ranging from 0 to 100, which represent the percentage of occurrence for a certain type of door. For example, an area with 50% in blue and 50% in purple will only contain blue and purple doors, and each will have the same chance of being blue or purple.

Because they are percentages, the sum of all weights on each area must equal exactly 100.

Advanced properties

These properties can be controlled from the windows that pop up from the Advanced settings buttons in the main window.

Excluded doors

Currently it's not possible to exclude doors from being randomized.

Initial pickups

You can choose which upgrades and expansions you'll start with upon landing on Tallon Overworld. Note that currently some pickups cannot be toggled on or off.

Missiles are expressed in total ammount of missiles, and the arrow buttons add or subtract 5 to that amount.

The maximum of each of the expansions is the same as the original game (250 missiles, 14 energy tanks and 8 power bombs).

Ignore the string of 0s and 1s at the end; it's for debug purposes and will be removed in upcoming updates.

Patch settings

These settings are for miscellaneous patches that are meant for difficulty settings and shortcuts for speedrunners.

Skipping

These settings are for skipping certain elements and sections of the game.

  • Skip item collected messages: The game will not pause to show a message when collecting an upgrade or expansion, and instead shows a small message at the top of the visor.
  • Skip Frigate: Skip the Pirate Frigate, with the game starting on the landing sequence to Tallon Overworld.
  • Skip Impact Crater: The warp to the Impact Crater is replaced to a warp to the end scene.

By default, only the Pirate Frigate is skipped.

Difficulty

These settings are for adjusting the difficulty of the game.

  • Heat damage protection: You can choose whether every suit except the Power Suit gives protection to Magmoor Caverns' extreme heat, or only the Varia Suit does.
  • Damage reduction: If you select "Always use the most powerful", you use the resistance given by the strongest suit you've collected, regardless of what suits you have. "Stack with suits" computes the total resistance given by each individual suit.
  • Power bomb lock-picking: (Not implemented yet) If you enable this option, you can use Power Bombs to open a door of any color.

By default it uses the same settings as the original game: Only the Varia Suit grants protection for extreme heat and the strongest suit's protection is always used.

Miscellaneous patches

These patches are merely aesthetical.

  • Fix Flaaghra music files: In the 0-00 version, the Flaaghra music's loop is broken, making it impossible to hear the whole theme. By using a Metroid Prime Trilogy disc file, it's possible to fix it so that the entire theme can be played.
  • Keep Attract Mode: If active, the videos that show on the title screen after a while will not be erased.
