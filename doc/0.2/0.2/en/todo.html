Coming soon in MPDR

This is a list of the planned features for the upcoming versions. (It also serves as a TODO for the developers!)

They are not ordered in any particular order.

Platform support

  • Support for mac OS from Sierra (10.12) onward.
  • Appimage for Linux, so that MPDR can be easily downloaded without building it from source.

Randomizer

  • Map patches: So that the doors' colors show up properly in the map.
  • Power Bomb lockpicking: In a pinch, use a Power Bomb to open a door of any color.
  • Easy Doors: When a non-blue door is opened, it will turn to blue permanently.

MPDR

  • Full multi-language support: Allows switching between languages without changing the system languages.
  • Door Excluder 2.0: Instead of just excluding doors from being randomized, also allow to force them being a different color.
