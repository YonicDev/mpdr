Stacking: Soporte para juegos modificados

Desde la versión 0.3, MPDR acepta imágnenes de disco modificadas de Metroid Prime. El proceso de aleatorizar una imagen ya modificada se denomina stacking.

Para hacer stacking de un juego aleatorizado con MPDR, simplemente escoge una ISO compatible, y MPDR debería reconocer el tipo de modificaciones realizadas y combinarlas.

Dependiendo de las herramientas usadas para crear estas imágenes modificadas, MPDR aplicará, importará del juego original, o ignorará algunos parches.

Compatibilidad de parches

Esta tabla muestra qué parches están disponibles para todas las herramientas oficialmente soportadas en la versión más reciente de MPDR:

  • ✓ Sí: Puedes usar MPDR para aplicar este parche en el juego modificado.
  • ◇ Importado: MPDR utilizará el parche equivalente del juego modificado.
  • - Ignorado: MPDR no aplicará este parche en el juego modificado.
  • ! Riesgo: MPDR podría no dar el resultado esperado cuando este parche sea aplicado, dependiendo de la configuración del juego modificado. También puede ser que haya alto riesgo de que el juego se cuelgue.
  • ⨯ No: MPDR no dará definitivamente el resultado esperado cuando este parche sea aplicado, o colgará el juego de manera casi constante.
