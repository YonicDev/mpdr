Novedades de MPDR v0.3

Estas son las principales características añadidas a MPDR versión 0.3 y sus respectivas subversiones.

Versión 0.3

Nuevo algoritmo RNG

Desde esta versión MPDR usa un generador de números aleatorio (RNG) parecido al randomizador de syncathetic. Esto hará que las semillas den resultados diferentes con respecto a las versiones anteriores.

Compatibilidad con otros randomizadores (Stacking)

A partir de esta versión, puedes utilizar una imagen modificada de Metroid Prime, permitiendo combinar los efectos de otro randomizador con los de MPDR. A esto se le denomina stacking.

Actualmente solo se soporta de forma oficial la versión 2.0 del aleatorizador de BashPrime.

Parche para el mapa

Ahora MPDR puede editar el mapa del planeta de forma que las puertas sean del tipo correcto. Esta funcionalidad se puede desactivar desde la ventana de Configuración de parches.

NOTA: Se ignorarán las puertas con escudos de misiles, y aparecerán con el color de la puerta que les corresponde, y no grises como en el juego original.

Randomizar puertas de una sola dirección

La puerta que conecta el Panteón con las Ruinas Chozo se puede convertir en una que puedas atravesar por los dos lados, permitiendo así que se pueda randomizar.

Para ello, activa la característica experimental "Randomizar puertas de una dirección" en la Configuración de parches, y en la ventana de Excluir puertas, activar la aleatorización de esta puerta.

Arreglos

  • Las puertas ya no tendrán una pequeña posibilidad de volverse azules cuando la influencia de las puertas azules es del 0%.
