No están ordenadas por prioridad o algún otra clase de orden particular.

Soporte de versiones

  • Soporte para Versión NTSC 0-01 de Metroid Prime.
  • Soporte para Versión PAL de Metroid Prime.

Randomizador

  • Super Puertas: Cuando una puerta no azul es abierta, se volverá azul permanentemente.

MPDR

  • Selección de idioma: Permite cambiar el idioma en el programa, y no teniendo que cambiar el idioma del sistema.
  • Editor gráfico de influencia: Una nueva forma de modificar la influencia del tipo de puertas usando un editor parecido a editar degradados.
  • Herramienta de exclusión de puertas 2.0: Que permitirá forzar que una puerta sea de un tipo determinado.
