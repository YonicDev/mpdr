Próximamente en MPDR

Esta es una lista de características planeadas para futuras versiones.

No están ordenadas por prioridad o algún otra clase de orden particular.

Soporte de versiones

ADVERTENCIA: Este soporte extendido de plataformas será temporal hasta que se trabaje en un método definitivo para combinar Metroid Prime (Item) Randomizer con MPDR.
  • Soporte para Versión NTSC 0-01 de Metroid Prime.
  • Soporte para imágenes de disco creadas con Metroid Prime (Item) Randomizer.

Soporte de plataformas

  • Soporte para mac OS a partir de Sierra (10.12).

Randomizador

  • Parches para el mapa que mostrarán los colores correctos de las puertas en todos los mapas.
  • Bomba Ganzúa: Activado esta opción, podrás usar una Bomba de Energía para abrir una puerta de cualquier color.
  • Super Puertas: Cuando una puerta no azul es abierta, se volverá azul permanentemente.

MPDR

  • Selección de idioma: Permite cambiar el idioma en el programa, y no teniendo que cambiar el idioma del sistema.
  • Herramienta de exclusión de puertas 2.0: Que permitirá forzar que una puerta sea de un tipo determinado.
